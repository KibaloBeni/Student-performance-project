# Student Performance Analysis – Python

A small data analysis project.

## Project summary
This project uses the **Student Performance** dataset from the **UCI Machine Learning Repository** to analyze factors associated with academic performance.

## Tools
- Python
- Pandas
- NumPy
- Matplotlib
- scikit-learn
- ucimlrepo

## What the notebook does
- loads a real public dataset
- explores the variables
- analyzes correlations with final grade
- creates clear visualizations
- trains a simple baseline regression model


## Run
Open the notebook and run:

```bash
pip install -r requirements.txt
jupyter notebook
```
